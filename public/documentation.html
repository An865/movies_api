myFlix API

This web application provides users with access to information about different movies, directors, and genres. Users can sign up, update their personal information, and create a list of their favorite movies.

myFlix URL Endpoints
Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
1. Get welcome message / GET None Text of welcome message 'welcome to myFlix'
2. Get list of ALL movies /movies GET None A JSON object holding array of all movies
3. Get list of ALL users /users GET None A JSON object holding array of all movies
4. Get movie data by title /movies/[name] GET None A JSON object holding data about movie
{
title: 'Psycho',
year: '1960',
director: 'Alfred Hitchcock',
genre: 'Horror'
}
5. Get user data by username /user/[name] GET None A JSON object holding data about user
{
Username: 'user1',
Password: 'password',
Email: 'user1@email.com',
FavoriteMovies: []
}
6. Get movie's genre by movie name /movies/[name]/genre GET None Text indicating the genre of the movie
7. Get director data by name /movies/[name]/director GET None Text indicating the director of the movie
8. Create new user /users POST A JSON object holding data about user to add {
Username: 'user1',
Password: 'password',
Email: 'user1@email.com',
FavoriteMovies: []
} 		A JSON object holding data that was added including id
{
_id: '1' Username: 'user1',
Password: 'password',
Email: 'user1@email.com',
FavoriteMovies: []
}
9. Add new movie to user's favorites /users/[name]/movies/[moviesid] PUT None Text of updated movie list
10. Update username /users/update/[oldName][newName] PUT None Text indicating the updated name of user
11. Delete movie from user's favorites /users/[name]/[movie] DELETE None JSON object holding user data that reflects deleted movie
{
Username: 'user1',
Password: 'password',
Email: 'user1@email.com',
FavoriteMovies: []
}
12. Delete user's account /users/[id] DELETE None Text indicating the user's account was deleted
