myFlix API

This web application provides users with access to information about different movies, directors, and genres. Users can sign up, update their personal information, and create a list of their favorite movies.

myFlix URL Endpoints
Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
1. Get welcome message / GET None Text of welcome message 'welcome to myFlix'
2. Get list of ALL movies /movies GET None A JSON object holding array of all movies
3. Get list of ALL users /users GET None A JSON object holding array of all movies
4. Get data about single movie by title /movies/[name] GET None A JSON object holding data about movie
{
title: 'Psycho',
year: '1960',
director: 'Alfred Hitchcock',
genre: 'Horror'
}
5. Get data about a genre by movie name /movies/[name]/genre GET None Text indicating the genre of the movie
6. Get data about director by name /movies/[name]/director GET None Text indicating the director of the movie
7. Create new user /users POST A JSON object holding data about user to add {
name: 'John Doe',
favoriteMovies: [Psycho]
} 		A JSON object holding data that was added including id
{
id: 2,
name: 'John Doe',
favoriteMovies: [Psycho]
}
8. Update user's favorite movie list /users/[name]/movies/[moviesid] PUT None Text of updated movie list
9. Update user info /users/update/[oldName][newName] PUT None Text indicating the updated name of user
10. Delete movie from user's list of favorites /users/[name]/[movie] DELETE None Text indicating movie was deleted from list of favorites
11. Delete user's account /users/[id] DELETE None Text indicating the user's account was deleted
